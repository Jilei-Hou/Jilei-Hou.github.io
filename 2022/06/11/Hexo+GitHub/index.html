    Hexo+GitHub搭建属于你的博客

    Hexo Hexo GitHub
    发布时间 : 2022-06-11 16:55
    字数:379 阅读 :
    1. GitHub创建仓库
    2. 安装Git及ssh key配置
    3. 安装Node.js
    4. 安装Hexo
    5. 插入图片

    哪个码农没想过拥有属于自己的个人博客呢。本文将详细介绍使用Hexo+GitHub搭建属于你的个人博客

    GitHub创建仓库

    相信大家都有自己的GitHub账户（毕竟全球最大的交友网站，哪个程序员不爱呢）

    新建一个GitHub仓库，仓库名应该为：用户名.github.io这个用户名必须是你的GitHub帐号名称（MUST）

    例如我的仓库名就是Jilei-Hou.github.io

    安装Git及ssh key配置

    安装Node.js

    Download | Node.js (nodejs.org)

    使用以下命令查看是否安装成功

    安装Hexo

    1. 安装
    npm install -g hexo-cli
    1. 初始化
    hexo init blog
    1. 新建文件
    hexo new test_my_site
    1. 生成（hexo g == hexo generate）
    hexo g
# 或者
hexo generate
    1. 启动服务预览（hexo s == hexo server）
    hexo s
# 或者
hexo server

    本地打开 http://localhost:4000/ 网页可以预览自己的博客（只可以本地哦，他人看不到）

    1. 部署（hexo d == hexo deploy）
    hexo d
# 或者
hexo deplo

    打开 https://用户名.github.io/ 网页可以预览自己的博客（只可以本地哦，他人看不到）

    例如我的博客网页就是：https://jilei-hou.github.io/

    插入图片

    hexo+github上传图片到博客 - 简书 (jianshu.com)

    安装如下插件

    npm install hexo-renderer-marked

    修改_config.yaml文件

    post_asset_folder: true
marked:
  prependRoot: true
  postAsset: true

    之后就可以在使用![](image.jpg)的方式愉快的插入图片了。
